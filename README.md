# 🧸 ToyVerse

A modern and responsive e-commerce web application for browsing, purchasing, and managing children's toys and products.

Built with React, Tailwind CSS, Axios, and TanStack Query while consuming the E-Comus REST API.

---

## ✨ Features

### 🛍️ Product Catalog

* Browse products from a live API
* Responsive product grid
* Product detail page
* Loading, error, and empty states

### 🔍 Search and Filtering

* Search products by name
* Filter by category
* Filter by brand
* Pagination support

### 🛒 Shopping Cart

* Add products to cart
* Update quantities
* Remove products
* Automatic total calculation
* Cart persistence after refresh

### 💳 Checkout

* Checkout form
* Order summary
* Submit orders to the API
* Success and error feedback

### 📦 Orders

* View order history
* Display order status
* Review purchased items

### 📱 Responsive Design

* Mobile-first design
* Tablet support
* Desktop support

---

## 🎨 Design System

### Color Palette

| Purpose      | Color     |
| ------------ | --------- |
| Background   | `#C9E4E7` |
| Cards        | `#FFFFFF` |
| Accent Color | `#ACECF7` |

The interface was designed to create a friendly and playful shopping experience suitable for parents and children.

## 🔄 State Management

### Server State

Managed using TanStack Query:

* Products
* Categories
* Cart
* Orders

### UI State

Managed using React state:

* Search input
* Current page
* Active filters
* Modal visibility
* Toast messages

---

## 🌐 API Integration

The application uses the E-Comus REST API.

Main resources used:

* Products
* Categories
* Cart
* Orders

Axios is configured using a centralized API client with interceptors and error handling.

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Create environment variables

Create a `.env` file in the project root:

```env
VITE_API_URL=your-api-base-url
```

### Start the development server

```bash
npm run dev
```
